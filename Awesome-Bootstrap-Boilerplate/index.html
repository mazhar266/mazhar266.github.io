Awesome

Bootstrap Boilerplate

.Start building your site from the template in 15 seconds. .

Awesome Bootstrap Boilerplate is here to boost the development of your new projects. It is a template based on HTML5 Boilerplate and Twitter Bootstrap. You will find font awesome and less all in the project folder. Just 2/3 lines to uncomment and it's all set.

 

What's included

  • HTML5 Boilerplate 4.3.0 - link
  • Twitter Bootstrap 3.2.0 - link
  • Font Awesome 4.1.0 - link
  • LESS.js 1.7.3 - link

 

Special Features

  • long text without whitespace link break fix
  • css3 font smoothing implemented
  • reset css of an element
  • selection protection css
  • smooth animation for mobile fix
  • hardware acceleration css class
